# rc-entry
This implements a way to join some campaign with `puppeteer`...

## Development
- Build
  - `yarn build`
- Watch code changes and keep building
  - `yarn build`
- Lint
  - `yarn lint`
- Test
  - `yarn test`

- Link
  - `yarn link`
     - Then `rc-entry` command can be used

## Usage
```
$ rc-entry --username 'YOUR EMAIL' --password 'YOUR PASSWORD'
```
